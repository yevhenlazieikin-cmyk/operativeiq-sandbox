How this sandbox works

Describe a page in the terminal on the left. Claude Code will read CLAUDE.md, pick the right skill (apply-code-design for plain-text descriptions, apply-figma-design for Figma URLs), and scaffold files under src/app/pages/. The new page shows up in the side navigation automatically once it's registered in app.routes.ts.

Available today

  • Full @backoffice/shared-ui public API
  • SCSS tokens — $primary-blue, $forest-green, $error-red
  • MessageService, permission guard stubs, resolver stubs
  • Mock JSON fixtures under src/app/__mocks__/
